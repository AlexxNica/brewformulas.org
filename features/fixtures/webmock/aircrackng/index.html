Description

Aircrack-ng is an 802.11 WEP and WPA-PSK keys cracking program that can recover keys once enough data packets have been captured. It implements the standard FMS attack along with some optimizations like KoreK attacks, as well as the all-new PTW attack, thus making the attack much faster compared to other WEP cracking tools.

In fact, Aircrack-ng is a set of tools for auditing wireless networks.

Under the spotlights

Channel -1 issues

If you are receiving an error message talking about channel -1, there are 3 solutions:

  • Patch you driver like explained in the wiki
  • Get the latest version of Aircrack-ng (1.2 Beta 1 or from our subversion repository) and use the option --ignore-negative-one.
  • Or you can compile aircrack-ng from source and use the flag libnl=true.
You can find an installation guide in the wiki.
